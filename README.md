## 성과

- FE
  - virtualize 를 별도로 구현 + 로딩시 스켈레톤 효과 추가(성능개선)
  - recoil 이용 무한스크롤링 구현
- BE
  - 별도 db 없이 파일을 검색하는 방식으로 구현
  - 파일 검색한 내용을 cache 에 저장 하고 반환하는 별도의 class 구현(같은 내용 검색시 캐쉬 사용)
